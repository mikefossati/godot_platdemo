# Quaternius Ultimate Platformer Pack

## About
This directory contains 3D models from the **Ultimate Platformer Pack** by Quaternius.

- **Source**: https://quaternius.com/packs/ultimateplatformer.html
- **License**: CC0 (Creative Commons Zero) - Public Domain
- **Usage**: Free for personal and commercial projects
- **Attribution**: Not required but appreciated

## Pack Contents (112 models)

### Character
- Animated character with 18 animations
- Rigged and ready for use
- Formats: GLB, FBX, OBJ, Blend

### Collectibles
- Coins
- Gems/Crystals
- **Stars** ⭐ (Using this for our game)
- Power-ups

### Platforms
- Grass platforms (various sizes)
- Stone platforms
- Wood platforms
- Modular pieces (corners, edges, centers)

### Environment (Not using yet)
- Trees
- Rocks
- Clouds
- Bushes

### Enemies (Not using yet)
- 4 unique animated enemies

## Directory Structure

```
quaternius_platformer/
├── character/          # Player character models + animations
├── collectibles/       # Stars, coins, gems
├── platforms/          # Platform models
└── environment/        # Trees, rocks, clouds (future use)
```

## File Formats

For Godot 4.5.1, we prefer **GLB** format:
- Best Godot support
- Smallest file size
- Includes materials and animations
- Single file per model
